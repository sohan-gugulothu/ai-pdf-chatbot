# AI PDF Chatbot

An AI-powered PDF chatbot built using Python and OpenAI APIs.

## Features
- Upload PDF documents
- Ask questions in natural language
- AI-generated responses
- Simple and clean UI

## Tech Stack
- Python
- Streamlit
- OpenAI API

- Chat history
